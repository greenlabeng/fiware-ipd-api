fiware-ipd-api

Install with:

    git clone https://github.com/greenlabeng/fiware-ipd-api.git
    cd fiware-ipd-api
    npm install

Start as follow:

    npm start

Test like this:

    curl http://localhost:3000/api/v1/devices
    curl http://localhost:3000/api/v1/devices/thing-px2/minst
